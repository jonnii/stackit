Make Stacked Changes Fast & Intuitive

A powerful CLI tool for managing stacked Git branches and pull requests. Work faster with better code organization.

Installation

Build from Source Recommended

Clone the repository and build using Go or Just:

# Using Go
git clone https://github.com/jonnii/stackit.git
cd stackit
go build -o stackit ./cmd/stackit

# Or using Just (if installed)
just build

Homebrew Coming Soon

Install via Homebrew (macOS and Linux):

brew install stackit

Binary Release Coming Soon

Download pre-built binaries from GitHub releases:

# Download for your platform
curl -L https://github.com/jonnii/stackit/releases/latest/download/stackit-[platform] -o stackit
chmod +x stackit

System Requirements

  • Go 1.25+ (for building from source)
  • Git 2.23+
  • GitHub CLI (optional, for enhanced PR features)

Quick Start

Get up and running with Stackit in minutes. Here's a typical workflow:

1

Initialize Stackit

Set up Stackit in your repository:

cd your-repo
stackit init
2

Create a Stacked Branch

Create a new branch on top of your current branch:

stackit create feature-part-1
# Make your changes
git add .
stackit create feature-part-2
3

View Your Stack

Visualize your stacked branches:

stackit log
4

Submit Pull Requests

Push all branches in the stack and create/update PRs:

stackit submit
5

Restack After Changes

Automatically rebase branches when the base changes:

stackit restack

Core Commands

Stackit provides a comprehensive set of commands for managing your stacked workflow:

stackit init

Initialize Stackit in your repository and configure the trunk branch.

stackit create [name]

Create a new branch stacked on top of the current branch with your staged changes.

stackit log

Display a visual representation of your stacked branches and their relationships.

stackit submit

Push branches and create/update pull requests for the entire stack.

stackit restack

Rebase branches in the stack to ensure each has its parent in history.

stackit sync

Sync your local branches with remote changes and update your stack.

stackit checkout [branch]

Switch between branches in your stack with autocomplete support.

stackit merge

Merge approved pull requests and clean up merged branches.

stackit squash

Squash commits in the current branch while maintaining the stack.

stackit split

Split commits into separate branches for better organization.

stackit absorb

Automatically absorb uncommitted changes into the appropriate commits.

stackit info

Show detailed information about the current branch and its position in the stack.
